Cara Membuat Auto Feed Artikel dengan HTML dan JavaScript

Panduan sederhana untuk menampilkan artikel dari halaman lain dalam domain yang sama menggunakan teknik fetch, DOMParser, dan querySelector.

Konsep Utama

Apa Itu Auto Feed Artikel?

Auto Feed Artikel adalah teknik untuk mengambil daftar artikel dari satu halaman sumber, lalu menampilkannya kembali di halaman lain secara otomatis.

Dalam metode ini, kita tidak langsung mengambil database. Yang diambil adalah HTML halaman sumber, lalu JavaScript membaca bagian tertentu seperti gambar, judul, link, deskripsi, dan tanggal.

Tahap 1

Siapkan Halaman Sumber Artikel

Pertama, pastikan Anda punya halaman sumber artikel. Contohnya:

https://suksespintar.biz.id/2/articles

Di halaman tersebut, struktur artikelnya sebaiknya konsisten seperti ini: 

<section>
  <article>
    <img src="gambar.jpg" alt="Judul Artikel">

    <h4>
      <a href="artikel-detail.html">Judul Artikel</a>
    </h4>

    <p>Deskripsi singkat artikel...</p>

    <small>28 April 2026</small>
  </article>
</section>
Tahap 2

Siapkan Halaman Tujuan

Halaman tujuan adalah halaman yang akan menampilkan artikel hasil tarikan. Buat wadah kosong seperti ini: 

<div id="artikel">
    Loading artikel...
</div>

Nanti isi dari div ini akan diganti otomatis oleh JavaScript.

Tahap 3

Ambil Halaman Sumber dengan Fetch

Gunakan perintah fetch() untuk mengambil isi halaman sumber. 

const res = await fetch("https://suksespintar.biz.id/2/articles", {
    cache: "no-store"
});

const html = await res.text();

Perintah ini mengambil halaman artikel dalam bentuk teks HTML.

Tahap 4

Ubah HTML Menjadi Dokumen

Setelah HTML berhasil diambil, ubah HTML tersebut menjadi dokumen yang bisa dibaca JavaScript. 

const doc = new DOMParser().parseFromString(html, "text/html");

Dengan DOMParser, kita bisa mencari elemen di dalam halaman sumber seolah-olah halaman itu sedang terbuka di browser.

Tahap 5

Ambil Semua Artikel

Setelah menjadi dokumen, ambil semua elemen artikel.

const articles = doc.querySelectorAll("section article");

Artinya JavaScript akan mencari semua elemen:

<section>
  <article>...</article>
</section>
Tahap 6

Ambil Gambar, Judul, Link, Deskripsi, dan Tanggal

Dari setiap artikel, ambil bagian pentingnya.

const img = a.querySelector("img");
const link = a.querySelector("h4 a");

const title = link.textContent.trim();
const href = link.href;
const desc = (a.querySelector("p")?.textContent || "").slice(0,120) + "...";
const date = a.querySelector("small")?.textContent || "";
Tahap 7

Tampilkan ke Halaman Tujuan

Setelah data diambil, susun menjadi card artikel.

output += `
<div class="card-artikel">
    <img src="${img.src}" alt="${title}">
    <div>
        <h3><a href="${href}">${title}</a></h3>
        <p>${desc}</p>
        <small>${date}</small>
    </div>
</div>`;

box.innerHTML = output;
Full Script

Contoh Script Lengkap

<div id="artikel">Loading artikel...</div>

<script>
async function loadArtikelProfil(){
    const url = "https://suksespintar.biz.id/2/articles";
    const box = document.getElementById("artikel");

    try{
        const res = await fetch(url, { cache: "no-store" });
        const html = await res.text();

        const doc = new DOMParser().parseFromString(html, "text/html");
        const articles = doc.querySelectorAll("section article");

        let output = "";
        let count = 0;

        articles.forEach(a => {
            if(count >= 6) return;

            const img = a.querySelector("img");
            const link = a.querySelector("h4 a");

            if(!img || !link) return;

            const title = link.textContent.trim();
            const href = link.href;
            const desc = (a.querySelector("p")?.textContent || "").slice(0,120) + "...";
            const date = a.querySelector("small")?.textContent || "";

            output += `
                <div class="card-artikel">
                    <img src="${img.src}" alt="${title}">
                    <div>
                        <h3><a href="${href}">${title}</a></h3>
                        <p>${desc}</p>
                        <small>${date}</small>
                    </div>
                </div>
            `;

            count++;
        });

        box.innerHTML = output || "<p>Tidak ada publikasi.</p>";

    }catch(e){
        box.innerHTML = "<p>Gagal memuat data.</p>";
    }
}

loadArtikelProfil();
</script>

Catatan Penting

Syarat Agar Berhasil

  • Lebih aman jika sumber dan tujuan masih dalam domain yang sama.
  • Struktur HTML halaman sumber harus konsisten.
  • Kalau struktur artikel berubah, selector JavaScript juga harus disesuaikan.
  • Metode ini tidak membaca database langsung.
  • Yang dibaca adalah halaman HTML yang sudah jadi.

Kesimpulan

Kesimpulan Sederhana

Dengan HTML dan JavaScript, Anda bisa membuat sistem penarik artikel otomatis dari halaman lain dalam domain yang sama. Teknik ini cocok untuk:

  • Portal artikel
  • Halaman profil
  • Portal kategori
  • Website komunitas
  • Landing page yang ingin menampilkan publikasi terbaru

Intinya: satu halaman menjadi sumber konten, halaman lain tinggal menarik dan menampilkan.
